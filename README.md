🌌 SERAPHIC // V12 OMNI
A high-performance, minimalist music interface built for background multitasking and Picture-in-Picture (PiP) stability.

🛠 Key Features
Persistent PiP: Fixed the "Tiiny Host" bug where minimizing the app killed the audio stream.
V12 Omni Engine: Uses a dual-loop visualizer to keep the browser process "hot" even when the tab is hidden.
PWA Ready: Installable on iOS and Android via the sidebar.
MediaSession API: Full lock-screen and dynamic island support.

🚀 How to Use On All Devices
iOS (iPhone & iPad)
Note: Must use the Safari browser for full "web app" functionality. 
Open Safari and navigate to the desired web app.
Tap the Share button (square with an up arrow) at the bottom of the screen.
Scroll down and select Add to Home Screen.
Optional: Rename the app and tap "Add" in the top right corner.

Android
Note: Google Chrome offers the best experience for this method. 
Open Chrome and navigate to the web app.
Tap the three-dot menu icon in the top-right corner.
Select Add to Home screen or Install app.
Follow the prompts to name the app and tap Add.
The app will now appear on your home screen and in your app drawer, acting as a dedicated app
